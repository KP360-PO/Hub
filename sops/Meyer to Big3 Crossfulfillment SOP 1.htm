SOP : Meyer to Big3 Crossfulfillment

 

1.   Export Orders from Channel Advisor

o   Navigate to Fulfill > Orders

Apply filters:

Distribution Center: Meyer to Big3

Shipping Status: Unshipped & Partially Shipped

 

o   Select all orders > Click Export then choose the Meyer to Big3 export layout, then click Export.

2.   Identify orders for Crossfulfillment

o   Open the exported file.

o   Go to column F (Public Notes).

o   Filter to show only blank cells these orders have not yet been swept, and need be cross fulfill.

 

3.   Manually Item Fulfill selected orders in ZM

o   Copy and paste the orders into ZM for manual item fulfillment.

o   Priority: Always select Big3 supplier over KSI unless no other option.

o   For Pair Orders with Mixed Distribution Centers - Check the availability in Big3 for both SKUs by manual item fulfillment in ZM.

        If both SKUs are available at Big3 or KSI, then:

        Cancel the part initially assigned to Meyer.

        Fulfill both SKUs via Big3 so they ship together in one shipment.

        If one or both SKUs are unavailable at Big3:

        Check if both SKUs can be fulfilled in Meyer, ideally in the same warehouse.

        If yes, Cancel the initial order, then manually reorder both SKUs from Meyer so they can ship together.

 

4.   Evaluate Cost

If Big3 fulfillment cost is too high

        Check the NPM

If still profitable we can proceed with Big3.

If not profitable, or if the part is OOS at Big3 & KSI

        Open the order in Channel Advisor

        Change the Distribution Center back to Meyer.

        Click Save to revert to Meyer.

 

5.   Update Public Notes in CA

o   For every order, add a clear/descriptive note in the public notes section summarizing what was done (e.g. Fulfilled in Big3, Reverted to Meyer due to OOS, etc.)

 

 

6.   Updating tracking number of all shipped orders

        Navigate to Fulfill > Orders

        Apply filters:

                                         i.    Distribution Center: Meyer to Big3

                                        ii.    Shipping Status: Unshipped & Partially Shipped

                                      iii.    Public Notes: Is Not Empty

        Open each order with notes and check in Shipstation if it is already shipped, if Yes, update the tracking and mark as shipped in CA.
