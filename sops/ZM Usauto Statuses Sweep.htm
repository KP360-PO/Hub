SOP: USAuto ZM Order Status

 

To ensure all USAuto orders in ZM are reviewed based on order status. Resolved correctly to prevent delays, and duplicate orders.

 

1. Ready to Send (Order is queued and ready for PO submission.)

Orders should not remain in this status longer than 4 hours

Action:

        Identify orders in Ready to Send status

        If order is older than 4 hours, send PO immediately

        Monitor for confirmation email

 

2. Pending Confirmation (Quote ID Received)

Order has received a Quote ID but no supplier confirmation yet.

Action:

        Monitor Pending Confirmation orders

        If no confirmation is received after 1 hour: Call USAuto to follow up document call and outcome

 

3. Processing (Quote ID + Order ID Received)

Order has been received by the supplier and is being processed for shipment.

Action:

        No action required

        Continue monitoring

        Wait for tracking number to be provided

 

4. On Hold / Approved (Duplicate Orders)

Orders on hold pending approval, often due to duplicate order review.

Action:

        Review duplicate orders

        Confirm with customer (CX) if required

If customer confirms:

        Approve order via Helpdesk Mailbox

        Once status changes to Approved, send PO

        Add internal notes

 

5. Failed Orders Status

Order failed during USAuto purchase or processing.

Action:

Investigate failure by:

        Checking PFG site for stock availability

        Confirming if item is OOS or discontinued, if confirmed OOS or discontinued, Tag order for cancellation to review and to notify the customer.

        Order should receive email confirmation (Thank you order) within 2 hours after send PO or order placed in PFG (for manual orders)

 

Important:

        If no confirmation is received, wait 24 48 hours before processing a manual order. This is to avoid duplicate orders

        Monitor closely for possible duplicate orders and cancel immediately if detected

 

6. OOS / Partial OOS

Order is out of stock or partially out of stock.

Action:

        Go to ZM > USAUTO > Orders

        Filter orders to OOS

        Tag in SS all pending OOS or Partial OOS orders

 

7. Crossfill

Order is eligible for cross-fulfillment to Willcall.

Action:

        Verify that all orders are properly cross fulfilled to Will Call

If not:

        Manually item-fulfill the order to display available suppliers

        If Will Call is too expensive or item is confirmed OOS / Partial OOS for a pair:

        Reverse order back to USAuto : Go to Actions > Reverse to USAuto

 

Select option:

CLX SKUs → Select Single

PLX SKUs → Select Pair

        Run action and verify update
