SOP: ITEM RECEIVING IN NETSUITE

 

1. Go to the Pick List SharePoint to obtain the files needed for receiving.

Link: https://netorgft5128391.sharepoint.com/sites/Warehouse/Shared%20Documents/Forms/AllItems.aspx?id=%2Fsites%2FWarehouse%2FShared%20Documents%2FWarehouse%20Pick%20Sheets&p=true&ga=1

2. Select the assigned warehouse (e.g., New Jersey), then choose Will Call.

 

 

 

3. Download the pick list per supplier by clicking the three dots ( ) beside the supplier name and selecting Download. The file will be saved to your Downloads folder.

4. The downloaded file must be separated per batch or per PO number. This is a sample of the downloaded file, which currently contains 8 pages with all batches dated 01/02 combined into one PDF. To begin separating the file, click the printer icon located on the upper left of the screen.

5. Identify the batch or PO number CL260102D-NJ-2, which contains 2 pages. In the print window, set the destination to Save as PDF, then select Custom under Pages and enter 2 3 as the page range for this PO. Click Save and name the file using the PO number. Repeat this step for the other batches and other suppliers, such as TYC, EAGLE, and KSI.

6. Once all files have been separated and saved, go to NetSuite and navigate to Transactions > Purchases > Enter Purchase Orders > List.

 

7. In the PO Number field, enter the PO number excluding the batch suffix (for example, enter CL251229D-CA instead of CL251229D-CA-1, -2, or -3) so that all related PO batches will appear in the list.

8. Right-click the View link from the previous step and select Open in new tab. This will open the Purchase Order details. If the status shows Pending Receipt, it means the PO has not yet been received. To proceed with receiving, click the Receive button.

9. This screen will open once the Receive button is clicked. Enter the batch number in both the Memo field and the Reference PO # field. Select your name in the Received By field and set the Class to Will Call.

10. Carefully cross-check the items against the pick list to ensure that all items match. If all items on the pick list are correct and complete, make sure the Receive boxes are checked, indicating that the items have been received.

11. Navigate to the Files tab, then click the plus (+) icon to upload and attach the downloaded pick list file that corresponds to this batch.

 

12. This window will appear after clicking the plus (+) icon. Enter the file name and click Choose File to select the file you want to attach, then click Save.

13. After clicking Save, you will be returned to the previous screen where the batch number will now appear, indicating that the file has been successfully attached. Click Save again to complete the process.

14. Repeat the same process for all batches and suppliers. If there is any Out of Stock (OOS) item, it will be indicated in the Notes column before receiving. If you see an OOS note, click the Edit button before proceeding with the receiving process.

15. This screen will appear after clicking the Edit button. For any items marked as Out of Stock (OOS) and confirmed as not listed on the pick list, check the Closed box to indicate that the item was not received. Click OK, then Save, and proceed with receiving by clicking Receive.
