SOP:  Scrubbing

Tools:

·        PO Scrubbing Tool Processor (https://kp360-po.github.io/KPP/PO_Portal.html)

·        Scrubbing Excel File (Manual)

·        Warehouse Spreadsheet

Purpose: Additional step to check accuracy of order/SKU selection. To also sort out the orders per suppliers and warehouse and identify MIX and Multiple items.

 

Step 1: Download the inventory file per warehouse in ZenShip.

 

Step 2: Upload the PO File (from ZM) and WH inventory file to PO Scrubbing Processor Tool.

Note: Watch the tutorial for additional information

 

Step 3: Export the file and post in All Suppliers PO Spreadsheet

Step 4: Distribute the orders per supplier per warehouse.
