SOP: Dropship Errors & Wrong Item Fulfillment

 

Check for Dropship Errors, possible OOS orders and dropship orders (Meyer, Turn14, Hanson) accidentally went to 100Ryan Fulfillment. 

 

DROPSHIP ERRORS  

Process: 

  • Go to Channel advisor Fulfill Tab > Select Distribution Center (Meyer, Turn14 , Parts Authority Tonsa Ksource) > Select Shipping Status: Unshipped and Partially Unshipped > Fiter Flag  . Filter Selected should look like image below. 

 

  • Identify and resolve the error for each order. Below are the sample errors. 

 

o    Unknown Internal Error   Look out for big/bulky item as it requires LTL Shipment and need to manually order in Meyer Website. (Please refer to Meyer Manual order procedure) You can also check in Meyer Open Documents using the Channel Advisor OrderID as reference if there s an existing order, if so, just add a public note with the PO/ OrderID. 

 

o    Invalid name - The name provided was not valid.   Update the Name listed on the Shipping address, make sure it does not contain any special characters or accented characters such as  , , , , , . After the update, click on Retry Shipment. 

 

o    Cost Exceeds Price  The supplier cost or shipping cost exceeds the order price.

 

Identify the applicable scenario and follow the corresponding steps below.

 

Case 1: Zero ($0) Amount

Usually generated automatically by Amazon replacement orders

 

Action:

        Investigate by checking customer conversation if the RS is valid.

        If unsure, confirm with managers and/or with Helpdesk (HD) whether the RS is valid or invalid

If invalid RS:

        Post the order to All Suppliers

        Advise suppliers to cancel the order

        Document confirmation and action taken

 

Case 2: Exceeded Due to Expensive Supplier Cost

Supplier item cost is higher than expected

 

Action:

        Validate if there is a significant cost difference

        Check the supplier s website to confirm the quoted price

        Determine whether the order can still be fulfilled profitably

If fulfillment is not viable:

        Attempt cross-fulfillment with another supplier (if available)

        If confirmed that the seller price differs from the supplier website price.

        Post details to the RMA Prevention Channel to request update in pricing.

 

Case 3: Exceeded Due to Expensive Shipping Cost

Shipping cost causes the total cost to exceed order price

 

Action:

        Calculate shipping using NPM Calc

        Confirm whether dropshipping can proceed within acceptable cost

If not viable:

        Attempt cross-fulfillment with another supplier

If shipment qualifies as LTL or Oversize:

        Post the tracker to Meyer/T14 Required Update Tracker so we can post it to RMA Prevention Channel to request update.

 

 

o    CARB Restriction   Orders with this error cannot be fulfilled due to California Air Resources Board (CARB) restrictions and will be cancelled or send with JWL order (if Ebay order). 

 

o    We have received an indication from the partner that this shipment has been canceled  Investigate why the order was cancelled by checking the purchases mailbox/Meyer Website using the Channel Advisor OrderID. Possible reasons are the following OOS, Partial OOS, No Meyer route available and need approval for Third Party Carrier etc. Decision making is needed whether it s okay to proceed or for cancellation.

 

o    Address Line Exceeds Character Limit - The shipping address exceeds the system character limit and cannot be processed. Review the shipping address details, confirm whether the full address is entered only in Address Line 1 or if possible: Split the address between Address Line 1 and Address Line 2 then Save changes and retry shipment

 

 

 

WRONG FULFILLMENT 

Check for Unfulfilled Dropship Orders in 100Ryan Stock. Please note that (MLX, TLX, ALX and OLX skus) are not available in our Warehouse. 

 

To check: 

  • Go to Channel advisor Fulfill Tab > Select Distribution Center (100 Ryan Street (Existing Stock) > Select Shipping Status: Unshipped and Partially Unshipped > Ctrl+F (MLX, TLX, ALX and OLX).
