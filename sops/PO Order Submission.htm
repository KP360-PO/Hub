SOP: Order Submission

Tools:

        NetSuite, NetSuite Importing Tool (https://kp360-po.github.io/KPP/PO_Portal.html)

        Order File Generator (https://kp360-po.github.io/KPP/PO_Portal.html)

        TYC (https://www.tycamericas.com)

        EAGLE (https://www.regionmax.com/auth/login)

        Purchases Mailbox

Purpose: To submit orders to suppliers.

For TYC:

Step 1: Create the NetSuite and order file. Copy the details from the Order Number to Address Ship column. Put all the details including the PO Number and download the NetSuite file. Always make sure to match the count of items between the supplier s spreadsheet and the file you created.

Note: Submission of order is limited to 50 pcs only. If the number of items you have for a certain batch is more than 50, you must submit the order multiple times. Example, total pc count for TYC NJ is 120, you ll have 3 orders/file to submit. 1 for the first 50, then 2nd 50 and 3rd for the last 20.

 

Step 2: Using Ordering File Generator Tool, upload the NetSuite file and click the Download files button.

 

Step 3: Upload the PO file to TYC website. Make sure that the file from the Ordering File Generator is saved as CSV (Comma Delimited) to avoid error.

 

Step 4: For Delivery Option, please always select Pickup at Warehouse and select the TYC address based on the PO you re submitting, NJ is Monroe Township, NJ, GA is Suwanee, GA, IL is Aurora, IL and Cali is Fontana, CA. Once selected, clicked on save and continue.

 

Step 5: Click on Bill my Account, Save and Continue then fill out the details below. Once completed, click on Place Order and wait for the confirmation.

 

For EAGLE:

Step 1: Using the provided templare from EAGLE, create your PO order file. Unlike TYC, there s no limit for the item count for EAGLE. Even if the pc count is more than 50, you can create the file and submit the order once.

 

Step 2: Follow the screenshot below for Eagle order submission.

Ship To:

        EAGLE NJ: CARPARTS360.INC

        EAGLE CALI: CARPARTS360.INC (CA)

        Shipping Type: always WILL CALL

        Click Start.

Review for OOS or QTY Change or Incorrect Items, once done, click ADD TO MY CART

 

Always review the following details especially the pc count which must match with the count in EAGLE spreadsheet. Click submit order and wait for the confirmation.

 

 

For DEPO and KSI:

Step 1: Create the NetSuite and order file. Copy the details from the Order Number to Address Ship column. Put all the details including the PO Number and download the NetSuite file. Always make sure to match the count of items between the supplier s spreadsheet and the file you created.

 

 

 

Step 2: Using Ordering File Generator Tool, upload the NetSuite file and click the Download files button.

Step 3: Log in to NetSuite and upload the DEPO and KSI file created from NetSuite importing tool. Hover your mouse to Setup > Import/Export > Saved CSV Import

 

Step 4: Select Record Type: Purchase Order then select the Template you will use. Please use the right templates in order to import the data accurately.

Step 5: Upload your CSV file and click on next.

For Initial upload, Data Handling is always ADD then click Next.

 

Step 6: This part is already mapped, al you just must do is to change some details like Purchase Order: Receive By which is the date when the order will be processed by the supplier or pickup by our warehouse. Purchases Order: Date (Req) is the date when you are processing the PO (same day you re doing the process). Once reviewed, click on next then last page it to select Run

 

 

 

To Submit DEPO and KSI orders:

Step 1: After you upload the CSV file or the NetSuite file, it s time to send the order to the supplier (DEPO/KSI). Go to Transactions > Purchases > Enter Purchase Orders > List

Step 2: From this list, select View of the PO/Supplier you re submitting, on our example, it s DEPO NJ

Click Create New Action (+) > Email

Recipients Tab: Add the recipients below for DEPO. For KSI, add KSI folks by typing KS

 

Message Tab: Select the right message template depending on the supplier.

 

 

 

 

 

 

 

 

 

 

 

Attachments Tab: Upload the PO file from Ordering File Generator Tool (Pivoted file)

 

Step 3: Click on Merge and Send. Always double check your recipients, message template and attachments before sending.
