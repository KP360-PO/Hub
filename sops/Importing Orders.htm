SOP: Importing Orders

 

Suppliers: DEPO, TYC, EAGLE and KSI 

Tools: ShipStation and Zenmighty 

Purpose: To gather all recent unprocessed orders from the marketplace and Channel Advisor

IMPORTING OF ORDERS FROM SHIPSTATION TO ZENMIGHTY  

Step 1: Open ShipStation and run Update all stores and wait until the refresh icon stops rotating. Then wait 15 mins. before proceeding to step 2.  

Step 2: Go to Zenmighty> POs Tab > Pending > Select all> Select Action> Click Run Automation > click Play button > Run Action

Note: Only do this process once or else it will cause duplication. 

 

Step 3: After Run Automation , wait for all pending orders to be fulfilled.



Note: As much as possible, there should be no pending orders before proceeding.
