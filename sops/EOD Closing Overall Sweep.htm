SOP: Order Last Sweep

 

Thorough check all pending and unshipped orders before shipping cut-off times.

        Ebay / Walmart / BC - orders need to be shipped before 12 am EST

        Amazon - orders need to be shipped before 12 am PST / 3 am EST

 

ShipStation- Sweeping

 

1. Awaiting Shipment Begin by reviewing all unshipped orders in Shipstation, sorted by ship by today . Investigate why these orders haven t shipped, with a focus on will-call orders. Confirm whether the order was processed and why no label has been selected. Check status in Zenmighty and stock availability, ensure the order is processed, or manually order the part if there was an error in ZM item fulfillment. If necessary, generate advance labels and if the item is Out of stock, take the appropriate corrective action.

 

GENERATE ADVANCE LABELS

a.   Filter Orders Start by filtering the orders from the Awaiting Shipment section in Shipstation, using the Label Selected Tag

 

b.   Sort by Ship Date After filtering, sort the orders by Ship by Today and select all orders that need to be shipped today. Add the tag Advance Labels to the selected orders.

 

 

c.   Review Label Conduct a swift check on orders tagged with Advance Labels

 

d. Add Internal Notes Use the bulk action feature to add internal notes to the selected orders, marking them as Reprint to inform the warehouse that an advance label has been generated and can be reprinted if needed.

e. Check Zenship Verify the order status in Zenship to see if any orders were canceled or transferred due to backorder. Double check the Ship from Location to ensure the label reflects the correct information.

 

f. Generate the Label Once everything is verified, proceed to generate the shipping label for the selected orders.

 

2. On Hold Sort the on-Hold orders by ship by today which mostly focus on USAuto orders. Cross check the status in PFG to ensure it has been submitted to their system. If the order has shipped or has tracking information but isn t synching with Zenmighty, Shipstation and Marketplace, manually mark the order as shipped, ensuring all details arer accurate. Since this is the final sweep, attention to details is crucial to avoid any late shipments.

 

Channel Advisor Sweeping

1. Review orders due to Ship today Navigate to Channel Advisor > Fulfill Orders, filter by the distribution center and select the dropship supplier. Next, filter the shipping status to unshipped and partially shipped . Then, filter the estimated ship date up to the current date, ensuring you re addressing all orders that need to ship by today.

 

2. Investigate Shipping Delays For orders that haven t shipped yet, thoroughly review why the shipment hasn t been processed.

 

3. Handle Partially Shipped Orders For partially shipped order, particularly those involving multiple suppliers, verify that the unshipped item has been submitted or processed. Once confirmed, mark the entire order as shipped using the tracking number from the shipped item. Ensure this information is recorded in the multi-shipment tracker to notify the customer about the split shipment and provide tracking updates.

 

 

Marketplace- Sweeping

1. Ensure all unshipped orders are processing - After handling urgent orders (those sorted by ship today ) double-check that all remaining unshipped orders are being processed.

 

2. Match Unshipped order Counts Compare the unshipped order count between Marketplace, SS and CA. Ensure the numbers match across all platforms. If there s a discrepancy, investigate the orders in question and confirm whether they ve been processed correctly.

 

3. Final Sweep Make sure every unshipped order left has been accounted for, processed, or is awaiting tracking, to prevent any missed shipments or violations.
