SOP: FALSE STATUS

1. Login to ZM, go to Order → Items, set Import Status to False, then open the filter menu and select 150 under Per Page.

2. Scroll down and check for Will Call SKUs such as CLX-, PLX-, KLX-, and LLX-, then copy the corresponding order number (for example: 05-14089-26650).

3. Go to the All Supplier POs spreadsheet and search for the order number by pressing Ctrl + F. Paste the order number 05-14089-26650, then click Find All.

·         If the order number does not appear in the results, it means the order needs to be manually fulfilled.

·         If the order number appears in the results, go back to ZM and proceed with the next order number.

4. To manually fulfill the order, go to ChannelAdvisor, click the Fulfill tab, then select Orders.

 

5. Paste the order number 05-14089-26650 into the Search Order ID field and press Enter. You should see the order appear in the results. Right-click on the Order Number and select Open Link in New Tab.

 

6. Scroll down to the Item Details section, then right-click the SKU and select Open Link in New Tab.

 

 

 

 

7. Get the Partslink number from the SKU page by checking the Description section or using Ctrl + F to search for “partslink”; for example, the Partslink number is NI3030190.

8. Go back to ZM, navigate to the Inventory tab, click Search, and paste the Partslink number NI3030190 to view all available SKUs by supplier/warehouse (where 100 = 100Ryan, 15642 = KP360, 6325 = McDonough, and 333 = KP360).

 

 

9. Go back to ChannelAdvisor and check the shipping address to determine where the order should be fulfilled; for example, if the address is in TX, verify on the map and fulfill the order from IL.

10. Go back to ZM and review the available SKUs and warehouse locations for the Partslink number to determine where the order can be fulfilled.

11. Once the fulfillment location is confirmed, go back to All Supplier POs and input the order number, SKU, fulfillment warehouse, and customer name accordingly.

12. Copy the details entered in All Supplier POs to the IL Supplier spreadsheet since the order will be fulfilled in IL, then proceed to the next order number and repeat the steps.
