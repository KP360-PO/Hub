SOP: KEMA Sales Order and Item Fulfillment Upload (NetSuite)

 

This process explains how to prepare and upload the required files into the KEMA Sales Order and Item Fulfillment Processor in NetSuite.

 

Generate the Tracking File (ShipStation)

 

        In ShipStation, navigate to: Shipments > KEMA SO

        Apply the Order Date filter. Select all orders for the selected date range.

        Click Export Shipments.

o   Export Settings

        Which records would you like to export? Export all n orders in the current view

        How would you like to export these? Export shipment line items

        Export view: Export my current view

        Download and save the exported file.

 

 

 

Generate the PO Number File (NetSuite)

 

        In NetSuite, go to: Vendors > Purchases > Enter Purchase Orders > List

        Set the following filters:

o   View: PO Team Purchase Order Order Details

o   Type: Purchase Order

        (Optional) Adjust the date range to generate a smaller file.

        Click the Excel icon to export the PO file.

 

 

 

Upload Files to the Processor

 

        Upload the following into the processor:

o   Fulfilled SKUs file

o   Tracking file

o   PO Number file

        Manually add the PO Number if required.

        Confirm all files are successfully uploaded.

 

Verify Processor Output

  • Once all files are uploaded, the processor should populate the KEMA Sales Order table.
  • Review the data to ensure:
    • Orders match across all files
    • Tracking numbers and PO numbers are properly linked

 

(Reference: final processor layout should look like this.)

 

        Add a file name then click Download Report. Change file type to CSV.

 

 

KEMA Sales Order Upload (NetSuite)

 

        Once the Sales Order CSV file is ready, go to NetSuite.

        Navigate to: Setup > Import/Export > Saved CSV Imports

 

 

        Under Filters, change Record Type to Sales Order.

        Select KEMA SO Template.

 

Upload Steps

 

        Upload the prepared CSV file.

        Click Next.

        Mapping is already configured click Next, then Run to complete the upload.

 

 

 

KEMA Item Fulfillment (NetSuite)

 

Generate Sales Order Reference File

 

        In NetSuite, go to:
Sales > Enter Sales Orders > List

        Set the following:

1.     View: For SO Search with details (Wholesale)

2.     Customer: Sub Of

        Click the Excel icon to export the file.

 

Note: This file is required to retrieve the Document Number, which is needed to generate the Sales Order Number for Item Fulfillment.

 

 

Item Fulfillment File Preparation

 

        Return to the KEMA Sales Order and Item Fulfillment Processor.

        Upload the exported Sales Order file.

1.   The processor will automatically populate:

        SO Number

        Document Number

        Apply filters to hide unnecessary columns.

        Enter a file name, then click Download Export.

        Change the file type to CSV.

 

Item Fulfillment Upload (NetSuite)

 

        In NetSuite, go to: Setup > Import/Export > Saved CSV Imports

        Under Filters, change Record Type to Item Fulfillment.

        Select KEMA SO Item Fulfillment v2.

        Mapping is already configured click Next, then Run to complete the upload.
