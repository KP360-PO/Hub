SOP: SKU Selection / Review

 

Suppliers: WH, DEPO, TYC, EAGLE and KSI 

Tools: ZenMighty 

Purpose: To thoroughly review available SKU options to match customer s order

Note: Each warehouse will have their own suppliers and folders.

        NJ 100Ryan, DEPO_NJ, TYC_NJ, EAGLE_NJ and KSI_NJ

        CALI CARTPARTS360_CA, DEPO_CA, TYC_CA and EAGLE_CA

        GA CARPARTS360_GA, DEPO_GA, TYC_GA and KSI_GA

        IL CARPARTS360_IL, DEPO IL, TYC_IL and KSI IL

Step 1: POs Tab > POs > Open supplier folder (one at a time) > Start reviewing and selecting correct item/s for the order.

 

Step 2: Supplier s Folder > Click on the Order Number column once to sort the orders (PAIR and MULTIPLE ITEMS will be arranged next to each other) to easily review pair items.

Step 3: Always double check PAIR and MULTIPLE ITEMS and make sure all items are POed in one warehouse.

How to spot PAIR and MULTIPLE items (You can also do CTRL + F and search for PLX and MULTI):

PAIR: SKU starts with PLX-

MULTIPLE: Order Detail column shows MULTI

Additional Note: Look for REPLACE to see if an alternative SKUs that were selected are correct.

Step 4: After reviewing all the orders and SKUs, Select ALL > Select Action > Processed > Run Action Button

 

Step 5: Once all the folders are checked and reviewed, POs > Select 1 folder only > Select Action > Send PO > Run Action Button

 

 

 

 

 

 

Fill out the highlighted fields below:

FILE NAME: CLYYMMDDSUPPCODE-WH-BATCH (example: CL260119E-NJ-1)

PO NUMBER: Same with the file name, CLYYMMDDSUPPCODE-WH-BATCH (example: CL260119E-NJ-1)

PO TYPE:

        NJ SUPPLIERS: Use Will Call for all suppliers EXCEPT KSI_NJ (use Drop Ship)

        CALI SUPPLIERS: Use Will Call for all suppliers EXCEPT CARPARTS360_CA (use Drop Ship)

        GA SUPPLIERS: Use Drop Ship for ALL SUPPLIERS

        IL SUPPLIERS: Use Drop Ship for ALL SUPPLIERS

PO STATUS: Select Send To Vendor

Once filled out, click Run Action button.

 

Step 6: Send Internal The purpose is to extract the orders from ZenMighty and put it in our online spreadsheet in preparation for PO submission to our suppliers.

POs > PO STATUS > OPEN AND SEE ALL > SELECT ALL FOLDERS PER WAREHOUSE THAT YOU PREVIOUSLY SENT TO PO

Sample below shows all CALI warehouse that were sent to PO earlier.

Send Internal Save to File > Run Action Button

Fill out the highlighted fields below:

FILE NAME: CLYYMMDDALL-WH-BATCH (example: CL260115ALL-CA-2B)

Once filled out, click Run Action button.

 

Step 7: Extract the file by downloading. POS > File Manager > Search for the ALL FILE per warehouse > Click the file > Copy URL > Open New Tab (browser) > Paste URL the file will be downloaded automatically.
